# Language-Translator-Program

## Description
- A C#/.NET console application that translates the phrase "Good Morning" into three different languages. This project was created to practice conditonal logic, user input handling, and basic program structure.

## Technologies Used
- C#
- .NET
- Visual Studio
  
## Features
- Language translation.
- Accepts user input to select a language.
- Console-based interface.
- Input validation to handle invalid selections.
  
## What I Learned
- Handling and validating user input in C#.
- Debugging logic errors during developement.
